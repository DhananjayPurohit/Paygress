About this dashboard

This page reads snapshot.json, a versioned and reproducible view of the marketplace produced by paygress-snapshot from live Nostr relays. The score column comes from the Sybil-resistant receipt aggregator; today the public corpus is empty, so all rows show 0.0 until receipts are posted.

Regenerate locally: cargo run --release --bin paygress-snapshot. Operators can run the same binary themselves and reproduce the published numbers with the same inputs (the aggregator core is pure — same offers + heartbeats + receipts in, byte-identical JSON out).
